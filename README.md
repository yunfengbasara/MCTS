# MCTS
中国象棋残局解法，通过Monte Carlo tree search（蒙特卡洛搜索树简称MCTS）的一种解决方法。残局来自天天象棋游戏残局楚汉争霸的第120关（乌江自刎），天天象棋游戏内点提示会提示一步棋，兵四进一，和MCTS解法一样。在MCTS内部，中国象棋棋盘采用二维坐标系（第一象限），所以MCTS内的第一步是（5，6）到（5，7）。
